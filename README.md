# cube_launcher 魔方桌面

## 介绍

3D的魔方桌面，使用应用图标作为方块，在玩弄（旋转、扭动）魔方的同时，点击图标打开相应的App。

支持：  

* 自定义配置魔方方块
* 魔方配色 
* 修改壁纸
